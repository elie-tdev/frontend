# Right Sidebar Pattern for MUI

Compatible with `@mui/material@^5` and `design-system/shared-frontend@^2`.

A high-level abstraction to handle right-hand-side drawer throughout an application. It has been designed and developed by the BAI XD team to serve as an example, and in many cases, a useable component OOTB for many teams.

## Documentation:

For further documentation, the `design-system/pattern-right-sidebar` package has it's [own section in our wiki](https://github.com/swt-xd/Shared-Frontend/wiki/Right-Sidebar-Home).
